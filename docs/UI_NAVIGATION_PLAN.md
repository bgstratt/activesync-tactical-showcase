## Landing Page Sections

- Hero: product statement and launch quick actions
- Architecture snapshot: runtime layers and data flow
- Feature matrix: link to each core facet page
- Mode gallery: link to each mini-game mode
- Live status strip: peers, ops/sec, queue, CAS hit ratio

## Sidebar Behavior

- Sticky left navigation visible on all non-embedded pages
- Route grouping and active section highlighting
- Collapsible groups for compact mode
- Search/filter over route titles

## Shared Page Frame

- Header with feature title and current session context
- Main interaction canvas
- Right diagnostics rail with:
  - convergence indicators
  - operation counters
  - merge/conflict events
  - replication latency chart
